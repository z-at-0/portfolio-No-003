# Portfolio No‑003

## 概要
このサイトは、ポートフォリオサイトです。  
閲覧者が直感的に情報を得られるよう、シンプルかつ見やすいレイアウトを心がけています。

## 目標・こだわり
- シンプルで見やすい情報設計
- ユーザーが直感的に操作できる UI
- モバイルにも対応したレスポンシブデザイン

## 主な機能
- トップページでの直感的なナビゲーション
- 各コンテンツへのリンク・詳細ページへの遷移
- モーダルウィンドウでの拡大表示（必要に応じて）

## 技術スタック
- 使用言語: HTML, CSS, JavaScript
- ライブラリ: なし（純粋な静的サイトとして作成）
